# М'ясоварня — сайт з кошиком і замовленнями в Telegram

## Важливо перед запуском
- **Логотип** (`public/assets/logo.png`) я вирізав із вашого скріншота Instagram, тож він
  трохи розмитий через низьку роздільність. Якщо у вас є оригінальний файл лого
  (без стиснення) — просто замініть цей файл своїм, назва має лишитись `logo.png`.
- **Ціни на бастурму, ковбаси і хрушки** — orієнтовні, я не бачив на них ваш реальний
  прайс (тільки на джерки). Відкрийте `data/products.json` і виправте на свої.

## Що вже працює
- Каталог товарів (редагується в `data/products.json`, без правок коду)
- Кошик: додавання, зміна кількості, видалення, підрахунок суми
- Підписка на товар (мінімум 1 місяць) — оформлюється як окрема позиція в кошику
- Форма оформлення замовлення → замовлення надсилається вам у Telegram
- Оплата: заглушка. Клієнт бачить повідомлення "ми зв'яжемось щодо оплати".
  Реальну оплату через Monobank Acquiring додамо наступним кроком.

## Як запустити локально
1. Встановіть Node.js (18 версія або новіша) — https://nodejs.org
2. У папці проєкту виконайте:
   ```
   npm install
   ```
3. Скопіюйте `.env.example` у файл `.env`:
   ```
   cp .env.example .env
   ```
4. Заповніть у `.env` свій `TELEGRAM_BOT_TOKEN` і `TELEGRAM_CHAT_ID` (як отримати — нижче)
5. Запустіть сервер:
   ```
   npm start
   ```
6. Відкрийте у браузері http://localhost:3000

## Як отримати TELEGRAM_BOT_TOKEN і TELEGRAM_CHAT_ID
1. Напишіть у Telegram боту **@BotFather** команду `/newbot`, дайте боту ім'я —
   отримаєте токен виду `123456789:ABCdefGhIJKlmNoPQRsTUVwxyZ`. Це і є `TELEGRAM_BOT_TOKEN`.
2. Напишіть своєму новому боту будь-яке повідомлення (наприклад "привіт") — це важливо,
   бот не може написати першим.
3. Відкрийте в браузері (замість TOKEN — ваш токен):
   `https://api.telegram.org/botTOKEN/getUpdates`
4. У відповіді знайдіть `"chat":{"id": 123456789, ...}` — це число і є `TELEGRAM_CHAT_ID`.

## Як редагувати асортимент
Відкрийте `data/products.json` і додайте/змініть/видаліть товар за зразком:
```json
{
  "id": "унікальний-ідентифікатор",
  "name": "Назва товару",
  "category": "Категорія",
  "weight": "100 г",
  "price": 150,
  "description": "Короткий опис.",
  "subscription": { "available": false }
}
```
Щоб товар можна було оформити як підписку — поставте `"available": true` і вкажіть
`"pricePerMonth"`. Зміни з'являться на сайті одразу після збереження файлу (перезавантажте сторінку).

## Як розмістити сайт онлайн (хостинг)
Найпростіше — **Render.com**:
1. Створіть безкоштовний акаунт на render.com
2. Завантажте цей проєкт у GitHub-репозиторій (або підключіть напряму)
3. У Render: New → Web Service → підключіть репозиторій
4. Build Command: `npm install`, Start Command: `npm start`
5. У розділі Environment додайте `TELEGRAM_BOT_TOKEN` і `TELEGRAM_CHAT_ID`
6. Після деплою отримаєте посилання виду `https://your-app.onrender.com`

Домен (наприклад `myasovarnya.ua`) можна купити окремо і прив'язати пізніше — 
це не заважає запустити сайт вже зараз.
